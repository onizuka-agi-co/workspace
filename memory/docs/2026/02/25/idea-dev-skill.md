# idea-dev スキル開発

## 背景
メンション時に現状把握ができるようにするため、新規企画開発専用のスキルを作成。

## 内容

### 検討した構成
1. **複数スキル案** — 読み込み用 + 保存用
2. **統一案** — 1つの `idea-dev` スキルで完結 ← 採用

### ワークフロー
```
1. 現状把握 (memory_search)
2. アイデア議論
3. メモリー保存（リクエスト時）
```

### メモリー保存フォーマット
```markdown
# トピック名

## 背景
## 内容
## 決定事項
## 次のステップ
```
