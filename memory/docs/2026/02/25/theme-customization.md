# テーマカスタマイズ

VitePressのカラーテーマをMaterial Design風にカスタマイズ。

## カラーパレット

| カラー | 用途 |
|--------|------|
| `#F24738` | メインアクセント（リンク、ボタン） |
| `#F2785C` | ホバー状態 |
| `#400707` | アクティブ、ダークモード |
| `#BF8275` | 補助色 |
| `#F2DC99` | ライト、グラデーション |

## 適用箇所

- リンク・ナビゲーション
- ボタン（brand, sponsor）
- サイドバーのアクティブ項目
- カスタムブロック（tip/warning/danger）
- ホームヒーローのグラデーションタイトル
- ダークモード対応

## ファイル

`docs/.vitepress/theme/custom.css` にCSS変数を定義。
